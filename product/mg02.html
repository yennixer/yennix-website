MG02 Multi-Spring Component Seal

Model: MG02 | Default Code: MG02-A-V-L

The MG02 is a multi-spring component seal with extended installation length, sharing the same face geometry as the MG01. Designed for applications requiring longer seal chambers, it provides the same reliable multi-spring performance with additional flexibility in installation.

Key Features

  • Multi-spring design for uniform face loading
  • Extended installation length variant
  • O-ring secondary seal
  • Same face geometry as MG01
  • Compatible with longer seal chambers

Applications

Chemical processing, Petrochemical, Water treatment, General industrial pumps
